# 개요

-   진행 기간 : 2024.02.19 ~ 2024.04.03

###

-   소모아 서비스 앱
    -   여러 제조사의 다양한 기기에서 사용되는 소모품을 관리해주는 앱
    -   제휴 쇼핑몰을 통한 자동 구매 지원
    -   소모품 상태에 따른 실시간 알림 지원

###

-   기기
    -   특정 제조사에 구애 받지 않고 여러 제조사 기기 등록을 지원함을 보여주기 위해 만든 테스트 웹 페이지

###

-   쇼핑몰
    -   제휴 쇼핑몰과의 연결을 보여주기 위해 만든 테스트 웹 페이지
